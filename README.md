# Suha Rehma Psychology

Marketing website for Suha Rehma, an evidence-based psychology practice. Tagline: "Mental Health is a Skill."

Live: **https://suharehma.vercel.app**

## Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 3.4
- Framer Motion 12
- Lucide React icons

## Sections

- **Hero** — Animated background circles, "Mental Health is a Skill" headline, Calendly booking CTA
- **Services** — Free 15-min Discovery Call + INR 1,000 / 50-min Therapy Session
- **Insights ("The Kahneman Library")** — 10 editorial cards on cognitive psychology topics (System 1/2 thinking, PTSD, burnout, neuroplasticity, etc.)
- **Writings ("From the Substack")** — 3D circular carousel of live posts pulled from [@neurodiversetherapist](https://neurodiversetherapist.substack.com) via RSS. Cards click through to Substack articles.
- **Crisis Center** — Three Indian mental health hotlines with click-to-call (Vandrevala Foundation, AASRA, iCall)
- **Footer** — Lao Tzu quote, booking CTA, social + Substack links

## Substack auto-refresh

Posts in the Writings carousel are sourced from `src/data/substack-posts.json`, which is generated by the fetch script:

```bash
npm run fetch-substack    # pulls latest RSS → src/data/substack-posts.json
```

A GitHub Actions workflow (`.github/workflows/refresh-substack.yml`) runs automatically on the **1st of every month at midnight UTC** — it fetches the RSS, commits the updated JSON, and Vercel redeploys. You can also trigger it manually from the **Actions** tab in GitHub.

## Setup

```bash
npm install
npm run dev              # Vite dev server
npm run build            # tsc -b && vite build
npm run fetch-substack   # refresh Substack posts data
```

No environment variables, no backend, no database. Static site only.
